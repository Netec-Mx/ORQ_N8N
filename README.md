# Orquestación y Automatización con N8N para Integración con IA



## Plataforma de laboratorios



Te damos la bienvenida a la **plataforma de laboratorios** del curso **Orquestación y Automatización con N8N para Integración con IA**. Aquí podrás explorar diferentes tecnologías a través de prácticas guiadas. ¡Desarrolla tus habilidades y lleva tus conocimientos al siguiente nivel!



## Lista de laboratorios



Cada uno de estos laboratorios está diseñado para ofrecerte una experiencia práctica. Haz clic en los enlaces para comenzar.


## Capítulo 1: Fundamentos de Orquestación con N8N

### [Práctica 1: Crear un flujo webhook → API externa → transformación → respuesta JSON](Capitulo_1/Lab1.md) 

  - **Descripción**: Construir un flujo automatizado con webhooks en n8n. Integrar APIs REST externas y servicios en Python. Transformar y combinar datos para generar respuestas dinámicas.

  - ⏱️ **Duración estimada**: 90 min.

## Capítulo 2: Integración de IA en Flujos Automatizados

### [Práctica 2: Flujo automatizado: formulario → análisis con modelo LLM (Bedrock o Foundry) → almacenamiento en base de datos local](Capitulo_2/Lab2.md)

  - **Descripción**: Construir la segunda fase del caso FinanzaPlus para que una solicitud de crédito se capture desde una interfaz web en Flask, se envíe al workflow existente de n8n, se enriquezca con análisis de riesgo usando un modelo desplegado en Azure Foundry y otro en Amazon Bedrock, y finalmente se almacene cada resultado en la base de datos local levantada con Docker.

  - ⏱️ **Duración estimada**: 90 min.
